# 🏗️ Build-Safe Connector

> **공정 지연 방지를 위한 데이터 기반 대체 자재 및 최적 공급망 추천 B2B 플랫폼** > 본 프로젝트는 건설 현장의 자재 수급 위기를 실시간 위치 기반 가중치 매칭과 이벤트 기반 역경매 알고리즘으로 해결하는 B2B 플랫폼입니다.

---

## 📌 프로젝트 개요

실시간 공공 API 호출의 한계를 극복하고 안정적인 서비스를 제공하기 위해 **'주기적 데이터 적재(Batch Layer)'**와 **'실시간 로컬 매칭(On-Demand Service Layer)'** 구조를 분리하여 운영합니다. 건설사에게는 선제적 리스크 모니터링을, 자재 공급처에게는 최적의 유휴 재고 매칭 및 간소화된 입찰 UX를 제공합니다.
